The `bigrams.zip` file contains two files:

* `tth.txt`, which is the full text (in ASCII) of Edgar Allen Poe's
  _The Tell-tale Heart_
* `coc.txt`, which is the full text (in ASCII) of H.P. Lovecraft's
  _The Call of Cthulhu_.

The data is adapted from the corresponding ebook at Project Gutenberg.
